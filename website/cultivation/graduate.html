研究生培养

生命科学技术学院“神经影像与睡眠机制研究组”研究生培养方案

一、研究方向简介

“神经影像与睡眠机制研究组”（以下简称睡眠组）立足多学科交叉，融合生物学、医学、信息科学、数学以及心理学等多学科知识背景，以脑电信号记录、核磁共振成像、正电 子发射断层扫描、行为多维生物电记录为主要研究手段，并借助基因检测、生化分析等现代生物学技术，针对睡眠神经科学领域的关键问题和前沿热点开展科学研究与 攻关。目前从事的研究方向包括：

  1. 睡眠的中枢神经机制研究
  2. 睡眠干预与人脑认知研究
  3. 睡眠信号自动划分阶段算法研究
  4. 睡眠监控与调节设备研发
  5. 针刺对睡眠的调控研究

二、培养目标

  1. 具备神经科学、电子技术、计算机技术以及信息科学有关的多学科交叉基础理论背景。
  2. 具备阅读外文科研文献的能力，并达到对文献进行系统总结、提炼及撰写综述的水平；能够进行小型学术报告。
  3. 培养“提出问题、解决问题”的科研思维及相应能力，具备从事科学研究的基本素质。
  4. 具备团队协作精神及良好的身心素质。

三、培养对象

本培养方案适用于“睡眠研究组”硕士一年级、非直博的博士一年级及实验室访问学生。方案中所列课程与考试以上学生必须参加，考试结果与当年奖学金直接挂钩， 同时学生的培养方案以各科考试成绩为基础，因材施教，突出个性。对跨学科背景较大的学生，可以在本方案的基础上做适当的调整。本课程方案为学校选课之外另加 课程，如有相同科目，可只修一次。考试形式分为：笔试，答辩和即时阅读演讲。考试由秦老师直接负责。

课程说明：

  1. 《生物学基础》主要以《探索脑》教材为主，老师讲课以报告形式进行，分为10到12次。报告前，探索脑前7章必须自学完成；《数学基础》课程，主要以统计和 磁共振数据处理为主；《信息科学应用》以报告形式为主，主要涉及睡眠相关内容。
  2. 本课程具体上课时间视当年学校上课时间而定，无故不得缺席。上课地点为G420教室、G421或G449会议室。
  3. 《精读文献》课程将由本组老师指定文献进行精读。每位学生精读十篇，全部制作ppt，其中3篇要求口头报告，每次报告时间不少于30分钟。其余7篇写出书面文 字报告。
  4. 《文献综述》课程，要求根据老师设定题目，自行查找文献、设计实验方案，最后形成ppt，进行小组汇报，小组全体老师打分。
  5. 自学课程以组内指定的四本国外优秀英文教材为主，通过自学-讨论-考核的方式完成课程学习任务。

五、考试

  1. 除《文献导读》之外，每门课程都需参加考试，并给以相应成绩（百分制）。
  2. 考试形式包括试卷考试、大报告以及平时成绩综合。关于大报告、综述等考试形式要求独立完成，严禁相互抄袭。
  3. 考试时间可在课程结束时，也可安排在其它相关课程学习之后。由各位老师自行决定。
    4. 注：直攻博一年级学生也需参加本方案所有课程与考试。非直攻博一年级博士生根据专业背景可对本课程方案做适当调整。

  2. Practical approach to EEG;
  3. Signal Processing for Neuroscientists;
  4. 《探索脑》
  5. 《认知神经科学-关于心智的生物学》
    6. 注：以上书籍都有电子版，需要的同学可以在朱元强博士那里登记，统一打印。自学书籍每2年会调整一次。
